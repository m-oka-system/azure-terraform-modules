# バリデーションチェックリスト

## 構造チェック（6 項目）

| #   | チェック項目                 | 期待値       |
| --- | ---------------------------- | ------------ |
| 1   | `variables.tf` の存在        | 必須         |
| 2   | `<モジュール名>.tf` の存在   | 必須         |
| 3   | `outputs.tf` の存在          | 必須         |
| 4   | `main.tf` が存在しないこと   | 禁止ファイル |
| 5   | `data.tf` が存在しないこと   | 禁止ファイル |
| 6   | `locals.tf` が存在しないこと | 禁止ファイル |

## variables.tf スタイルチェック

| #   | チェック項目 | 詳細                                                                                                              |
| --- | ------------ | ----------------------------------------------------------------------------------------------------------------- |
| 1   | 最小宣言のみ | `variable "xxx" {}` の形式。`type`、`description`、`default` が含まれていないこと                                 |
| 2   | 並び順       | `common` → `resource_group_name` → `tags` → `random`（必要時） → 主リソース変数 → 従属リソース変数 → 外部依存変数 |
| 3   | 空行         | 各 variable ブロック間に空行が 1 行あること                                                                       |

## `<モジュール名>.tf` スタイルチェック

| #   | チェック項目         | 詳細                                                                                      |
| --- | -------------------- | ----------------------------------------------------------------------------------------- |
| 1   | リソースラベル       | すべてのリソースのラベルが `this` であること                                              |
| 2   | ループ               | `for_each` を使用し、`count` をループ目的で使用していないこと                             |
| 3   | セクション区切り     | リソースグループ間に `################################` を使用していること                |
| 4   | 命名規則             | `<CAFプレフィックス>-${each.value.name}-${var.common.project}-${var.common.env}` パターン |
| 5   | セキュリティコメント | セキュリティ設定に日本語コメントがあること                                                |
| 6   | data source 配置     | data source がファイル先頭に配置されていること                                            |
| 7   | tags 配置            | `tags = var.tags` がリソースブロック末尾に配置されていること                              |

## outputs.tf スタイルチェック

| #   | チェック項目     | 詳細                                                                                   |
| --- | ---------------- | -------------------------------------------------------------------------------------- |
| 1   | リソース全体出力 | `value = azurerm_xxx.this` の形式でリソースオブジェクト全体を返していること            |
| 2   | 個別属性なし     | `value = azurerm_xxx.this.id` のような個別属性の output がないこと                     |
| 3   | output 名        | リソースの論理名（リソースタイプから `azurerm_` プレフィックスを除いた名前）であること |
